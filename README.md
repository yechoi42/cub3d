# cub3d

my first raycasting engine 

## basic mode

```
$ make
$ ./cub3d ./maps/basic.cub
```

![스크린샷 2020-08-01 오후 9.50.53](/Users/yechoi/Desktop/스크린샷 2020-08-01 오후 9.50.53.png)

- textured wall
- sprite
- wall collision
- object collision
- screenshot ( `./cub3d ./maps/basic.cub --save`)

## bonus mode

```
$ make bonus
$ ./cub3d ./maps/pengsoo.cub
```

![스크린샷 2020-08-01 오후 9.58.05](/Users/yechoi/Desktop/스크린샷 2020-08-01 오후 9.58.05.png)

- minimap
- lifebar
- background music
- items(tuna cans)
- looking down and up (↑ and ↓ key)
- change speed( + and - key)
